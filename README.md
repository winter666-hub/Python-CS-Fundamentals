### Python
- 클래스와 객체 (OOP)

- 내부 클래스

- 해시 테이블, 트리, 그래프
- 선형 탐색, 이진 탐색
- 버블 정렬, 선택 정렬, 삽입 정렬, 퀵 정렬, 병합 정렬 등

### Data Visualization
- Matplotlib (꺾은선, 막대, 산점도, 히스토그램, 파이차트 등)
